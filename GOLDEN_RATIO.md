# 황금 비율 얼굴 정의

황금 비율(Golden Ratio) φ ≈ **1.618**을 기반으로 얼굴의 4가지 비율을 측정하여 판별합니다.

---

## 측정 항목

### 1. 얼굴 종횡비 (Face Aspect Ratio)

```
얼굴 길이(이마~턱) / 얼굴 너비(광대) ≈ 1.618
```

- 얼굴이 적절히 길고 갸름한 형태인지 판단
- 너무 둥글거나 너무 길쭉한 얼굴은 해당 안 됨

### 2. 얼굴 3등분 균등비 (Facial Thirds)

```
상부(이마~눈썹) : 중부(눈썹~코끝) : 하부(코끝~턱) ≈ 1 : 1 : 1
```

- 얼굴이 수직으로 세 구간으로 균등하게 나뉘는지 판단
- 비율이 균등할수록 황금 비율에 가까움

### 3. 눈 간격비 (Eye Spacing Ratio)

```
양 눈 사이 거리(inner-to-inner) / 한쪽 눈 너비 ≈ 1.618
```

- 두 눈 사이가 한쪽 눈 크기와 비례하는지 판단
- 눈이 너무 붙거나 너무 벌어진 경우 해당 안 됨

### 4. 코-입 너비비 (Nose-to-Mouth Width Ratio)

```
입 너비 / 코 너비 ≈ 1.618
```

- 코와 입의 너비 비율이 황금 비율인지 판단

## 판별 기준

```
황금 비율 φ = 1.618
허용 오차 = ±5% (기본값, config.py에서 조정 가능)
허용 범위 = 1.618 × (1 ± 0.05) = [1.537, 1.699]
```

- 활성화된 측정 항목이 **모두** 허용 범위 내에 있을 때 → 황금 비율 얼굴 ✅
- 하나라도 범위를 벗어나면 → 모자이크 처리 대상 ❌

각 측정 항목은 `config.py`에서 개별적으로 ON/OFF 할 수 있습니다.
